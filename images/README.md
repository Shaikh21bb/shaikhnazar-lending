# Папка для изображений проектов / Project Images Folder

Сюда клади скриншоты своих работ для раздела "Портфолио".

## Как добавить проект:

1. Положи скриншот сюда (например: `my-project.jpg` или `my-project.png`)
2. Открой файл `js/portfolio.js`
3. Добавь новый объект в массив `PORTFOLIO_PROJECTS`:

```js
{
  title: "Название проекта",
  description: "Короткое описание",
  tags: ["Лендинг", "AI"],
  image: "images/my-project.jpg",   // ← имя файла из этой папки
  link: "https://your-project.com", // ← URL или "" чтобы скрыть кнопку
},
```

## Рекомендации по скриншотам:
- **Размер:** 1280×720 или 1920×1080
- **Формат:** JPG (сжато), PNG или WebP
- **Соотношение сторон:** 16:9 — выглядит идеально в карточке
